# Tarô 78 Chaves - Kit do Testador Beta (Versão 1.0.1)

Olá, estudante! Você foi selecionado para testar a versão Beta 1.0.1 do app Tarô 78 Chaves.
Seu feedback é fundamental para garantirmos uma experiência premium e sem falhas.

## URL de Acesso
https://taro78chaves.lovable.app

## Checklist de Teste
Por favor, realize as seguintes ações e verifique se tudo ocorre conforme o esperado:
1. [ ] Fazer Login (Email/Google)
2. [ ] Navegar pela Jornada (página inicial)
3. [ ] Abrir uma Lição (ex: Arcano O Louco)
4. [ ] Verificar progresso (deve iniciar em 0/78)
5. [ ] Acessar Ferramenta de Tiragens
6. [ ] Acessar Dicionário de Combinações
7. [ ] Acessar Perfil e Customer Portal (Stripe)
8. [ ] Testar navegação em dispositivo móvel

## Como enviar Feedback
Utilize o nosso canal oficial (ex: WhatsApp do suporte ou Formulário Beta) enviando:
- **Print da tela** onde ocorreu o problema.
- **Rota/URL** (ex: /app/licao/louco).
- **Descrição objetiva** do erro ou comportamento estranho.

## Classificação de Feedback (Interna)
Seu relato será classificado pela equipe em:
- Bug Crítico (impede o uso)
- Bug Visual (erro de layout ou texto)
- Problema Mobile (dificuldade no celular)
- Problema de Conteúdo (erro no texto do Tarô)
- Problema de Pagamento (Stripe/Assinatura)
- Dúvida de Uso (não entendeu como funciona)
- Sugestão Futura (ideia para novas versões)

Agradecemos sua participação!
